Loader

Use a loader within any space to display loading in progress. It can be anything, loading an image, some data, etc.

Toggle Loader Here

Usage

  1. Include the following code within any block element and the loader will overlay the entire width and height of its container.
  2. Also Include Script loader.js on your page.
  3. Initialize the loader using loader.init(id);. The id should match the loader's id.
  4. Use the following methods:
    • You can show the loader using loader.show(id);
    • You can hide the loader using loader.hide(id);
    • You can toggle the state of the loader using loader.toggle(id);
toggle markup 
<div id="myLoader" class="loader loader-overlay">
	<div class="loader__load-box"></div>
</div>
