Navigation

A navigation list can be used to provide links to major sections of the page.

Other Variations - Usage

Apart from just aesthetic classes, if you would also like your navigation list to stick to the top on page scroll, you can add data-nav-type="sticky" to your nav list and include a script sticky-nav.js (mandatory).

You may optionally add more details like:

  1. the scroll content ID (if you have a main content) using data-content-id="contentId",
  2. the header ID (if you have a site <header>) using data-header-id="headerId", and
  3. the footer ID (if you have a site <footer>) using data-footer-id="footerId"

Have a look at the code below.

toggle markup 
<!-- Sticky Navigation -->
<ul class="nav nav--light" data-nav-type="sticky" data-content-id="contentId" data-header-id="headerId" data-footer-id="footerId">
	<li class="nav__item nav__item--active"><a class="nav__link" href="#">Home</a></li>
	<li class="nav__item "><a class="nav__link" href="#">About</a></li>
	<li class="nav__item "><a class="nav__link" href="#">Projects</a></li>
	<li class="nav__item "><a class="nav__link" href="#">Contact</a></li>
</ul>

Scrollspy

You can make a navigation list a scrollspy navigation list which will highlight the section you are at.

Any navigation list, horizontal or vertical, can be made into a scrollspy.

You can see a vertical scrollspy working here on the showcase website.

Usage

In order to make a navigation list into a scrollspy, you need to mandatorily add the following attributes to it.

  1. Add data-scrollspy to let the system know that the nav list will be a scrollspy,
  2. Add data-spy-on="main-content" to let the system know the main content ID
  3. Wrap your entire content area inside <div id="main-content"> (note the same ID in step 2), and
  4. Add data-nav-type="sticky" to the nav list so that it sticks within the viewport.
  5. Include Script sticky-nav.js
  6. Include Script scrollspy-nav.js
toggle markup 
<ul class="nav" data-scrollspy data-spy-on="main-content" data-nav-type="sticky" >
    <li class="nav__item nav__item--active"><a class="nav__link" href="#">Home</a></li>
    <li class="nav__item "><a class="nav__link" href="#">About</a></li>
    <li class="nav__item "><a class="nav__link" href="#">Projects</a></li>
    <li class="nav__item "><a class="nav__link" href="#">Contact</a></li>
</ul>

Scrollspy (Progress Style)

Create a Progress type Scrollspy on top of your page. It will tell you how much of the page has been seen through and how much is left.

This can be very useful especially for blogs where, there are long articles, as this will tell how much the article has already been read.

To see a working progress type scrollspy, click here.

Usage

In order to make create a progress type scrollspy, you need to mandatorily add the following attributes to it.

  1. Include the following code given below within your HTML markup. You should ideally have it placed as the first child of your <body> tag.
  2. Wrap your entire content area inside <div id="main-content"> (note the ID, it should match the value of data-spy-on attribute), and
  3. Include Script scrollspy-progress.js
toggle markup 
<div class="scrollspy c-progress c-progress--rounded" data-spy-on="main">
    <div class="c-progressbar__meter progress-animate" data-animprogress="true"
        role="progressbar" aria-valuenow="0" aria-valuemin="0" aria-valuemax="100">
        <span class="c-progress__label"></span>
    </div>
</div>
